The Token Approval Helper for PPAS ("the Extension," "we," "our") is a Chrome browser extension that helps authorized administrators automate token approval processes on the PPAS portal at ppas.pdsodisha.gov.in. This privacy policy explains how the Extension handles data and protects your privacy.

4. How the Extension Works

The Extension operates entirely within your browser:

  1. Uses Your Existing Session: The Extension uses your existing PPAS login session (browser cookies) to authenticate API requests. It does NOT access, store, or transmit your credentials.
  2. Sends Approval Requests: When you click "Start Approval," the Extension sends POST requests to ppas.pdsodisha.gov.in on your behalf.
  3. Processes Responses Locally: Server responses are parsed locally in your browser. No data is sent to external servers.
  4. Displays Results: The Extension shows progress and results directly in the popup interface.

6. Permissions Explained

The Extension requests the following Chrome permissions:

6.1. "scripting" Permission

Why needed: To inject automation code into PPAS web pages for interacting with token approval forms.

Privacy impact: This permission allows the Extension to read and modify content on ppas.pdsodisha.gov.in pages only.

6.2. "activeTab" Permission

Why needed: To access the currently active PPAS tab when you click action buttons.

Privacy impact: This permission only grants temporary access when you explicitly trigger an action.

6.3. "storage" Permission

Why needed: To save your preferences locally.

Privacy impact: Data is stored only on your device and never synchronized or transmitted.

6.4. Host Permission

Permission: "https://ppas.pdsodisha.gov.in/*"

Why needed: To send API requests to PPAS portal endpoints.

Privacy impact: This restricts the Extension to ONLY work on the PPAS portal.
