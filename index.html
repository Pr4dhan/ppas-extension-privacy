PPAS Token Approval Extension

This extension is NOT officially affiliated with PPAS, Government of Odisha, or any government entity. It is an unofficial, community-created tool designed to assist authorized PPAS administrators.

1. Overview

The PPAS Token Approval Extension ("the Extension") is designed to help authorized administrators automate the token approval process on the PPAS portal (ppas.pdsodisha.gov.in). This privacy policy explains how the Extension handles data.

2. Data Collection

We DO NOT collect any personal data.

The Extension does not:

  • ❌ Collect, store, or transmit any personal information
  • ❌ Track your browsing activity
  • ❌ Access your PPAS credentials
  • ❌ Store token IDs or approval data
  • ❌ Send any data to external servers
  • ❌ Use analytics or tracking services

3. Local Storage

The Extension temporarily stores the following data locally in your browser only:

  • Selected expiry date (for current session)
  • Maximum token limit preference (for current session)
  • Start/stop state (temporary flag)

This data:

  • ✅ Never leaves your device
  • ✅ Is deleted when you close the browser or clear browsing data
  • ✅ Is only used to operate the extension

4. How the Extension Works

The Extension:

  • Uses your existing PPAS login session (browser cookies)
  • Sends approval requests to ppas.pdsodisha.gov.in on your behalf
  • Processes server responses locally in your browser
  • Does NOT intercept, store, or transmit your credentials

5. Permissions Explained

Why We Need These Permissions:

"scripting":

Allows the Extension to inject automation code into PPAS pages to interact with token approval forms.

"activeTab":

Allows the Extension to access the currently open PPAS tab to perform automation when you click "Start Approval".

"storage":

Allows the Extension to save your preferences (expiry date, token limit) locally in your browser.

"https://ppas.pdsodisha.gov.in/*":

Limits the Extension to only work on the official PPAS website, ensuring it cannot access other websites.
